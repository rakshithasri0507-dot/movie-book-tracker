# 🎬📚 Movie & Book Tracker

A full-stack web application to track movies and books you have watched or read.

## 🚀 Tech Stack

Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

---

## ✨ Features

- Add Movies
- Add Books
- Track Status (Completed / Pending)
- Store Data in Database
- REST API Integration

---

## 📦 Installation

Clone the repository:

git clone https://github.com/rakshithasri0507-dot/movie-book-tracker.git

Install backend dependencies:

cd backend
npm install
npm start

Install frontend dependencies:

cd frontend
npm install
npm start
